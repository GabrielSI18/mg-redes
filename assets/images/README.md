# Instruções para Adicionar as Imagens

Salve os seguintes arquivos de imagem nesta pasta:

1. **logo.png** - Logo da MG Redes (terceira imagem enviada)
2. **mecanico-interior.jpg** - Técnico trabalhando no painel do carro (primeira imagem)
3. **mecanico-caminhao.jpg** - Técnico trabalhando na cabine do caminhão (segunda imagem)
4. **caminhonete-equipamentos.jpg** - Caminhonete branca com equipamentos de climatização (quarta imagem)
5. **caminhao.png** - Imagem de caminhão para seção de aplicações versáteis

Após salvar as imagens com estes nomes exatos, o site estará completo e funcional!
