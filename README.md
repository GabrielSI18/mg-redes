# MG Redes Website

Este projeto é um site estático para a empresa MG Redes, baseado no material de referência fornecido.

## Estrutura do Projeto

- `index.html`: Arquivo principal com a estrutura do site.
- `style.css`: Arquivo de estilos CSS.
- `assets/images/`: Pasta para armazenar as imagens do site.

## Como Usar

1.  Abra o arquivo `index.html` em seu navegador para visualizar o site.
2.  Para adicionar as imagens reais:
    -   Coloque os arquivos de imagem na pasta `assets/images/`.
    -   Edite o arquivo `index.html` e substitua os caminhos das imagens nos locais indicados (procure por `<!-- Placeholder for ... -->`).

## Seções Implementadas

-   **Hero:** Logo e slogan.
-   **Quem Somos:** História e estatísticas da empresa.
-   **Nossos Serviços Principais:** Lista de serviços com destaque.
-   **Serviços Técnicos e Industriais:** Detalhamento técnico.
-   **Transporte e Logística:** Diferenciais logísticos.
-   **Energia e Telecomunicações:** Outras áreas de atuação.
-   **Por Que Escolher a MG Redes:** Diferenciais e contato.
